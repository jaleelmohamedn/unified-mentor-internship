# 💬 Chatapp – Real-Time Multi-Room Chat Application

A modern, responsive, **glassmorphism-styled** web-based chat platform built with **HTML, CSS, JavaScript, Node.js, and WebSockets (Socket.IO)**.  
Users can create and join chat rooms, send real-time messages, and enjoy a smooth and interactive chatting experience.

---

## 🚀 Features

### 🎨 User Interface
- Clean and modern **3D glassmorphism design** on all pages
- Fully responsive layout for mobile, tablet, and desktop
- Chat room interface with:
  - **List of available rooms**
  - **Message display area**
  - **Message input field**

### ⚡ Real-Time Communication
- Real-time messaging powered by **Socket.IO**
- Messages appear instantly without page refresh
- Multiple chat rooms supported simultaneously

### 🔐 User Authentication
- Choose a **unique username** before joining
- Prevents duplicate usernames and impersonation
- Room-specific user tracking

### 💬 Chat Features
- Send and receive text messages
- Display **username** and **timestamp** for each message
- Supports basic formatting:
  - **Bold:** `**text**`
  - *Italic:* `*text*`
  - Links: `https://example.com`

### 🏠 Room Management
- Create new chat rooms instantly
- Join existing rooms from a room list
- Room-based message history (stored in session)

### ✨ User Experience
- Auto-scrolls to the latest message
- Notifications for new messages
- Handles empty messages and invalid room/username input
