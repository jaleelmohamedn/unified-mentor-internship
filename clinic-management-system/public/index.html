Welcome

This web app helps doctors and receptionists coordinate tokens, patient records, prescriptions, and billing using Firebase.

  1. Receptionist: Create patients, generate tokens, capture basic info, and initiate billing.
  2. Doctor: View waiting list by token, add diagnosis & prescriptions, and update patient history.
  3. Audit: All actions are logged (time, user, action, payload) for traceability.
